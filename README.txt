1. 배경화면이 2초 간격으로 옆으로 슬라이드 되며 다음 화면으로 변경된다.
	1.1 슬라이드 기능은 라이브러리를 사용하여도 무방하다.
2. 글자는 상하좌우 마진이 동일한 가운데 정렬이다.
3. sns의 이미지는 font Awesome 라이브러리를 사용한다.
4. css 및 js는 외부에서 로드하는 방식으로 import 해야한다.
6. 참고 된 이미지를 보고 작업하며 슬라이드 이미지는 image 폴더에 있는 이미지를 사용한다.

html 파일은 한개만 존재해야 하며 이름은 index.html로 지정해야 한다.
css 파일은 여러개가 존재 할 수 있으며 사용처에 맞는 이름으로 지정한다.
js파일 또한 여러개 존재 할 수 있으며 사용처에 맞는 이름으로 지정한다.

완성된 작업물은 git에 업로드 하여 git 링크를 공유하여 제출한다.
